# MBTA Tracker App

## About the project
Don't miss the T! Boston T Station schedule tracker. 

## Getting started


### Installation
Go to project root directory, and run:
```sh
pip3 install -rrequirements.txt
```

### Run the tracker
Go to project root directory, and run:
```sh
# For led matrix
python3 -m src.mbta-tracker

# For simulation
python3 -m src.simulation
```
